### 2.3 生成

Luckyegg会生成以下生物：

- 骷髅
  - 掉落常规战利品+合金碎片（1％）
- 闪电苦力怕
  - 取消常规战利品，掉落火药＊5（不受抢夺）+随机唱片（10％，不受抢夺），+合金碎片（1％）。自爆无掉落
- 烈焰人
  - 常规战利品+合金碎片（1％）
- 凋零骷髅
  - 取消常规战利品，掉落凋零骷髅头（25％不受抢夺），+合金碎片（1％）
- 唤魔者
  - 常规战利品+合金碎片（1％）
- 幻术师
  -  随机附魔书（50％）+合金碎片（1％）

MOD还会生成以下掉落物：

- 1颗钻石（30%）
- 2颗钻石（30%）
- 3颗钻石（8%）
- 4颗钻石（2%）
